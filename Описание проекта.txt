Aegis AI Telegram Bot

Интеллектуальный ассистент для юридической компании "ИДЖИС". Бот интегрирован с CRM-системой через Firebase и использует Google Gemini API для общения с клиентами.

Функционал

AI-консультации: Автоматические ответы на вопросы по банкротству (БФЛ).

Синхронизация с CRM: Оператор может перехватить диалог в любой момент.

Распознавание контактов: Бот автоматически извлекает номер телефона из диалога.

AI-Саммари: Создание краткого резюме ситуации клиента для юриста.

Технологии

Node.js (node-telegram-bot-api)

Google Gemini AI (Generative Language API)

Firebase Firestore (Real-time DB)

Установка

Клонировать репозиторий.

Создать файл .env на основе .env.example.

Запустить npm install.

Запустить бота командой npm start.
